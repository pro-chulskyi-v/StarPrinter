## Подсказки | Hints
Возьмём matrixSize = 10 | We take matrixSize = 10
```plaintext
 for (int i = 0; i < matrixSize; i++)
 {
    for (int j = 0; j < matrixSize; j++>)
    {
        if (УСЛОВИЕ)
            Console.Write('*');
        else
            Console.Write(' ');
    }
 }
```
